# hakuX

hakuX is a xemu (X1 BOX) fork, optimized for Adreno devices.

## Setup Requirements

To use the emulator, you must provide your own legally obtained original Xbox files, such as:

- MCPX Boot ROM
- Flash ROM / BIOS
- Xbox hard disk image
- Game files/images from your own discs

hakuX does not include copyrighted BIOS or game content.
